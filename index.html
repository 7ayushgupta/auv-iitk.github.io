AUV IITK

Autonomous Underwater Vehicle
Sponsered by IITK powered by Passion.

AUV-IITK is a team of Undergraduate Students enthusiastic towards creating an efficient underwater vehicles which can get a wide array of tasks done. Alongside a wide array of industrial applications our designs can be used by NIOT for furthering their research on Autonomous Underwater Vehicles.

We aspire to overcome the challenges set for us.

NIOT SAVe has a set of tasks which have to be completed autonomously by the BOT. These are the goals that motivate us to work even harder.

  • Write all the code
  • Stack small boxes
  • Read books and stuff
  • Drink much coffee
  • Lightning bolt
  • Shadow clone technique
