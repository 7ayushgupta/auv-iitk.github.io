Controls

The team uses Odroid processor for processing of input from sensors and giving output to micro-processors. The different systems are connected to each other using efficient message passing on ROS.

Robot Operating System
The team writes all it's code in C++ which runs on ROS enabling us to connect the different systems efficiently. We also use handlers to seperate the different logical levels hence allowing us to efficiently implement control and intelligence algorithms.
Processor
We use an Odroid U3 processor with xUbuntu OS running on it to effectively use ROS. We will soon be upgrading to custom made i7 processors.
Simulation
We simulate given code with equations of fluid dynamics to find the maximum velocitie, drag co-efficients etc with given velocities of water and iterate this for various velocities to help optimize the structure of our Bot.
